DEV MODE
==

You have to
* Start the frontend server on a terminal window
* Start the backend server on another terminal window
* Then, go to http://localhost:8080


Frontend
===
Needs:
* npm
* yarn

```bash
cd web
yarn i
yarn serve

```

Backend
===

Needs:
* docker
* docker-compose


```bash

cd backend
docker-compose up
```
